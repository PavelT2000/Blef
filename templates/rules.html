♠ Правила «Блеф» ♣

Полное описание режима этой онлайн-версии

Суть игры

«Блеф» — карточная игра на одну колоду (52 карты) для 2–13 игроков. Вы не обязаны иметь заявленную комбинацию на руке: в свой ход вы называете покерную комбинацию, которая должна быть не слабее предыдущей заявки за столом. Соперники могут повысить ставку или сказать «Не верю» — тогда проверяется, есть ли названная комбинация среди всех карт всех активных игроков (вся рука стола).

Проигравший вскрытие получает дополнительную карту из колоды. Набрал лимит карт — выбываешь. Побеждает последний оставшийся игрок.

Комната и вход

  • На главной создайте комнату (6-символьный код) или войдите по коду / ссылке /join/КОД.
  • В лобби нужно минимум 2 игрока; старт может нажать любой участник.
  • На столе видны только свои карты; чужие отображаются рубашкой.
  • Ссылка «пригласить» на странице игры ведёт друзей в ту же комнату.
  • Комната без игроков или 5 минут без активности (входы, старт, ходы) удаляется автоматически.

Колода и карты на руках

  • Используется стандартная колода: ранги 2–10, валет (11), дама (12), король (13), туз (14); масти ♣ ♦ ♥ ♠.
  • В начале партии каждый получает по одной карте.
  • После каждого проигранного вскрытия проигравший берёт ещё одну карту из колоды (рука растёт).
  • После вскрытия, если игра продолжается, карты активных игроков перетасовываются из общей колоды с сохранением количества карт у каждого.
  • Если колода закончится, сервер вернёт ошибку — на практике лимит карт и число игроков подобраны так, чтобы колоды хватало.

Лимит выбывания

Число карт, при котором игрок выбывает, считается автоматически перед стартом:

  1. raw = ⌊52 ÷ число игроков⌋
  2. лимит = clamp(raw, 4, 12) — не меньше 4 и не больше 12
ИгроковПример лимита
2–412 карт
68 карт
105 карт
134 карты

В лобби отображается планируемый лимит до начала игры.

Ход игрока

Ходы идут по кругу среди не выбывших игроков. В свой ход можно:

  • Повысить / заявить — назвать комбинацию строго сильнее текущей ставки раунда (см. ниже).
  • «Не верю» — только если на столе уже есть ставка в этом раунде; оспариваете последнюю заявку.

На ход отводится 30 секунд (таймер на сервере). По истечении времени:

СитуацияАвтодействие
Уже есть ставка в раундеАвтоматически «Не верю»
Ставок ещё нетМинимальная заявка: Старшая карта 2

«Не верю» и вскрытие

  1. Оспаривается последняя заявка в раунде (кто её сделал — «заявитель»).
  2. Сервер проверяет, можно ли собрать эту комбинацию из всех карт на столе (у всех активных игроков вместе).
  3. Комбинация есть → не веривший ошибся → штрафную карту получает тот, кто сказал «Не верю».
  4. Комбинации нет → заявитель блефовал → штрафную карту получает автор последней ставки.

Экран вскрытия показывается 10 секунд; затем раунд завершается автоматически (можно нажать «Пропустить»). Карты активных игроков перераздаются из колоды, ставки раунда обнуляются, ход у проигравшего вскрытие (если он не выбыл) или у следующего активного.

Иерархия комбинаций

От слабой к сильной (повышение может быть на более высокий тип или на тот же тип с более сильными параметрами):

  1. Старшая карта
  2. Пара
  3. Две пары
  4. Сет (тройка)
  5. Стрит
  6. Фулл-хаус
  7. Флеш
  8. Каре
  9. Стрит-флеш
  10. Флеш-рояль

Параметры заявки

При повышении нужно указать тип и поля (подсказка на форме меняется под комбинацию).

КомбинацияЧто указыватьПроверка на столе
Старшая карта Достоинство (2–Т) Хотя бы одна карта этого ранга
Пара Достоинство Не меньше двух карт этого ранга
Две пары Два разных достоинства По две карты каждого ранга
Сет Достоинство Не меньше трёх карт ранга
Стрит Нижняя граница (вкл.) 5 карт подряд по рангу, младшая ≥ границы; колесо 2-3-4-5-Т при границе ≤ 2
Фулл-хаус Тройка + пара 3 одного ранга и 2 другого
Флеш Нижний ранг (вкл.), масть опц. 5 карт с рангом ≥ границы; при масти — все одной масти
Каре Достоинство Не меньше четырёх карт ранга
Стрит-флеш Нижняя граница (вкл.), масть опц. Стрит одной масти; при указании масти — именно она
Флеш-рояль Масть опц. 10, В, Д, К, Т; при масти — все этой масти
Нижняя граница включительна (≥). Для стрита, флеша и стрит-флеша выбранный ранг — это минимум младшей карты подходящей комбинации; карты рангом ниже в расчёт не идут. Для пары, сета, каре и старшей карты указывается точное достоинство, а не «от и выше».

Сравнение ставок одного типа

  • Старшая, пара, сет, каре — выше достоинство → сильнее.
  • Две пары — сначала старше старшая пара, при равенстве — младшая.
  • Фулл-хаус — сначала сравнивается тройка, затем пара.
  • Стрит, флеш, стрит-флеш — выше нижняя граница → сильнее.
  • Флеш, стрит-флеш, флеш-рояль — заявка с указанной мастью сильнее той же без масти.

Победа и выход

  • Остался один невыбывший игрок — он победитель.
  • Если во время игры активных стало меньше двух (все вышли), партия завершается с оставшимся победителем.
  • Покинуть комнату — вы удаляетесь из списка; при пустой комнате она закрывается.
  • Новая игра (лобби) после конца партии — те же игроки, сброс в лобби без смены кода комнаты.

Интерфейс за столом

  • В шапке: чей ход, таймер, размер колоды, текущая ставка.
  • Блок «Повышения в раунде» — кто и до какой комбинации повысил.
  • Свои карты открыты; при вскрытии и конце игры карты всех видны.
